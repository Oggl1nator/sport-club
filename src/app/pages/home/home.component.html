Добро пожаловать!

Вас приветствует коллектив спортивного клуба “Олимпиец”. Спортивный комплекс предоставляет услуги по проведению индивидуальных и групповых спортивных занятий.

Доступные на данный момент функции:

1. Авторизация и выход из приложения при помощи keycloak.

2. Просмотр списка всех существующих групп. Просмотр групп не требует авторизации.

3. Просмотр списка активных абонементов. Просмотр абонементов требует авторизации с ролью "Менеджер".

4. Просмотр списка тренеров. Также имеется возможность добавления нового тренера. Просмотр списка тренеров требует авторизации с ролью "Тренер" или "Менеджер". Для использования возможности добавления нового тренера пользователь должен быть авторизован с ролью "Менеджер".

5. Учет посещений тренировок спортсменами. К данному разделу имеют доступ только тренеры. Раздел предоставляет возможность выбрать тренера, затем из списка спортсменов, у него занимающихся, выбрать тех, у которых нужно списать из абонемента одно занятие. Если счетчик занятий стал равен нулю, абонемент аннулируется.
